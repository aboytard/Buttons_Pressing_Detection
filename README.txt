## Button Pressing Detection:
For the RET project, the button pressing Detection is meant to communicate the information about the button pressing. This package is meant to run with the ret package on the computer 'https://github.com/aboytard/ret'

## Requirement
Get the Button_Pressing_Detection repo  
`git clone https://github.com/aboytard/Buttons_Pressing_Detection.git`  

You may have to install influxDB, to do so use the same scheme than on the computer, don't forget that the RPI is working on arm64.

## Setup the communication
Setup the communication with the Computer currently configured on: 10.4.11.132

## Run the test
`sudo python ButtonPressingDetection/scripts/Button_Pressing_Detection_RET.py`
